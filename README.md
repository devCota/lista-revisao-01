## 🧩 Resolução das Questões

### **1. Resultado da execução do código**

```java
int x = 5;
int y = 2;
double resultado = x / y;
System.out.println(resultado);
```

**Resposta correta:**  
b) 2  

**Explicação:**  
A divisão é feita entre inteiros (`int`), então o resultado é truncado antes de ser atribuído ao `double`.

---

### **2. Saída do código**

```java
int x = 5;
System.out.println(x++);
System.out.println(++x);
```

**Resposta correta:**  
d) 6 e 7  

**Explicação:**  
x++ imprime 5 e depois incrementa para 6.  
++x incrementa para 7 e depois imprime.

---

### **3. Declaração correta de array**

**Resposta correta:**  
a) int[] arr = new int[5];  

**Explicação:**  
Forma padrão de declaração com tamanho definido.

---

### **4. Operador de concatenação de strings**

**Resposta correta:**  
a) +  

**Explicação:**  
O operador + concatena strings em Java.

---

### **5. Assinatura correta do método main**

**Resposta correta:**  
a) public static void main(String[] args)  

**Explicação:**  
Essa é a assinatura obrigatória para execução de programas Java.

---

### **6. Saída do código**

```java
String hello = "Hello";
hello += " World";
System.out.println(hello);
```

**Resposta correta:**  
c) Hello World  

**Explicação:**  
O operador += concatena a string.

---

### **7. Exceção não tratada**

**Resposta correta:**  
c) O programa gera uma exceção de tempo de execução.  

**Explicação:**  
Se não tratada, a exceção interrompe a execução do programa.

---

### **8. Criação de objeto**

**Resposta correta:**  
a) MinhaClasse obj = new MinhaClasse();  

**Explicação:**  
Uso do operador new para instanciar objetos.

---

### **9. Resultado da expressão lógica**

```java
boolean resultado = (5 > 3) && (2 < 4) || (6 >= 8);
```

**Resposta correta:**  
a) true  

**Explicação:**  
(5 > 3) → true  
(2 < 4) → true  
true && true → true  
(6 >= 8) → false  
true || false → true  

---

### **10. Valor da variável resultado**

```java
int a = 5, b = 3, c = 7;
boolean resultado = (a > b) && (b < c) || (c == a);
System.out.println(resultado);
```

**Resposta correta:**  
a) true  

**Explicação:**  
(a > b) → true  
(b < c) → true  
true && true → true  
(c == a) → false  
true || false → true
